# SnakeGame

贪吃蛇小游戏，使用原生 HTML、CSS 和 JavaScript 实现。

## Repository

- GitHub: https://github.com/DaPengRuYi/SnakeGame
- GitCode: https://gitcode.com/airuyi/SnakeGame

## Maintenance

主要维护 GitHub 仓库，同步推送到 GitCode 仓库。

推荐推送顺序：

```powershell
git push github main
git push gitcode main
```

## Run locally

直接用浏览器打开 `index.html`。
